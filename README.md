# GY271

## 简介

GY271 采用 HMC5843 芯片,HMC5843 采用霍尼韦尔的各向异性磁阻(AMR)技术，这一技术带来的好处是其他磁传感器技术无法企及的。这些传感器的主 要特点在于轴向高灵敏度和线性、固相结构、垂直轴间低灵敏度，用于测量地球磁场的方向和磁力，从十万分之几高斯到 6 高斯(gauss)。 霍尼韦尔的磁传感器位于行业内灵敏度最高和可靠性最好的低强度磁场传感器之列。



## 参数

输入电压: 3-5v.

通信方式: IIC 通信协议

测量范围: ±1.3-8Ga
